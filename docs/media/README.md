# Médias du README

Dépose ici tes images, **avec exactement ces noms de fichiers** — le README principal
les référence déjà, elles s'afficheront automatiquement :

| Fichier | Contenu suggéré |
|---|---|
| `hero.jpg` | Photo « héros » : le boîtier Poup allumé (OLED) **à côté du téléphone** affichant l'app. C'est l'image d'en-tête. |
| `hardware.jpg` | Le montage ESP32 + capteurs + OLED (de près, ou dans son boîtier près du vase). |
| `oled-demo.gif` | Petit GIF de Poup animé sur l'écran OLED quand on touche le capteur. |
| `app-home.png` | Capture de l'écran d'accueil de l'app (Poup + scène florale). |
| `app-history.png` | Capture de l'écran Historique (les graphes). |

Astuce : pour le GIF OLED, filme l'écran puis convertis en GIF (ou garde un .mp4 et
mets juste une capture). Formats : `.jpg`/`.png` pour les photos, `.gif` pour l'animation.
Vise < 2-3 Mo par image pour que la page GitHub reste légère.
